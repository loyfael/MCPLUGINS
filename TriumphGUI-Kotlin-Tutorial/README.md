# TriumphGUI Kotlin Tutorial

Tutoriel Kotlin minimal pour Triumph GUI sur Paper 26.1.2, Java 25 et Gradle 9.4.1.

## Versions

- Paper API: 26.1.2.build.72-stable
- Java: 25
- Kotlin: 2.4.0
- Triumph GUI: 3.1.11
- Shadow: 9.4.3

## Build Windows

```powershell
.\gradlew.bat clean build
```

Le JAR installable est généré dans `build/libs/TriumphGUI-Kotlin-Tutorial-1.0.0-SNAPSHOT.jar`.

## Contenu du tutoriel

- GUI simple
- navigation entre menus
- placement d'items
- gestion des clics
- fermeture d'inventaire
- liste dynamique
- composants Adventure pour les titres et lores
